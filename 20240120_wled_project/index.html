NOAH'S LED BOARD


Build a custom led board
Here i build a custom led (fully programmable IC LED'S) board. There are many LED'S on the market, however if you want to control each sinlge LED with a specific color, ending up in showing multiple colors (one per LED) through a effect, the LED's which are used must be adressable / IC leds. The well known addressable led's are the Ws2812b (RGB) led's, the more modern version are the so called Sk6812 (RGBW) which have an additional white led.
