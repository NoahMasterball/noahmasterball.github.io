Use the Rest API (via Javascipt) to change the world map into a daylight map
Before we use JavaScript we have to use the first API Commands to our WLED. Additional this Webpage contains the correct JavaScript Code to to send an API Command to the World Map (192.168.179.27) sending the correct settings via API for showing the correct world's daylight, based on the time when this page was loaded in our home network.
We can use a simple CLI Command named curl which retrieves the webpage (while retrieving the commands are hidden into the URL):
curl "http://192.168.179.29/win&T=1&A=5&FX=0&SX=0&R=0&G=255&B=238"

Or we use the browser which retrives the Webpage (while retrieving the commands are hidden into the URL):
http://192.168.179.29/win&T=1&A=5&FX=0&SX=0&R=0&G=255&B=238

E.g.: Following command got sent here: T=1 which means "Led Power on!"
http://192.168.179.29/win&T=1&A=5&FX=0&SX=0&R=180&G=215&B=230

E.g.: Following command got sent here: A=128 means brightness to 128"
http://192.168.179.27/win&A=128

E.g.: Following command got sent here: PL=1 means to turn on the Preset with the Quick Load Number 1"
http://192.168.179.27/win&PL=1

Example from the JSON API where the URL is never changing:
curl -X POST "http://192.168.179.27/json/state" -H "Content-Type: application/json" -d {"on":true,"bri":128,"transition":7,"mainseg":0,"seg":[{"id":0,"start":0,"stop":133,"grp":1,"spc":0,"of":0,"on":true,"bri":255,"col":[[255,136,0,0],[0,0,255,0],[0,21,255,0]],"fx":0,"sx":128,"ix":128,"pal":0,"sel":true,"rev":false,"mi":false},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0},{"stop":0}]}

For JSON (Javascript Object Notation) we usually use Postman to send the JSON command to the API.
In case you start programming with an API sending JSON Data to the API i highly recommmend using the software Postman, where you can test and train your API Commands.
